## Running the sample 

1. Build the server

~~~
npm install
~~~

2. Run the server

~~~
npm start
~~~

3. Go to local [http://localhost:8080](http://localhost:8080)
4. Go to public [https://nghehuavan8080.loca.lt](https://nghehuavan8080.loca.lt)

5. Special setup with http-proxy-cli forward http from port 8081 to 8080
~~~
npm run proxy
~~~
